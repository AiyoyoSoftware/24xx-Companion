# 24xx Companion

A web companion app for running solo or GM-less 24xx RPG games.

**[Play Here](https://aiyoyosoftware.github.io/24xx-Companion/)**

## About 24xx

24xx is a rules-light tabletop RPG system by Jason Tocci that uses dice pools for resolution. This **community made** companion app helps you manage a character, roll dice, track resources, and use oracle tables for solo or GM-less play.

## Features

### Character Management
- **Three Origins**: Create Human, Android, or Alien characters
- **Skills System**: Track skills with die sizes (d6, d8, d10, d12)
- **Inventory & Loadout**: Manage credits, armor, and gear
- **Character Traits**: Customize with specialties and alien traits

### Dice Rolling
- **Skill-based Rolls**: Roll dice pools based on your character's skills
- **Modifiers**: Add helping dice or hindered conditions
- **Roll History**: Track all your rolls in a persistent journal

### Data Management
- **Customizable Content**: Edit skills, gear, and oracle tables
- **Persistent Storage**: All data saved locally in your browser
- **Export/Import**: Backup and restore your custom lists

## Tech Stack

- **React 19** with React Router
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **Zustand** for state management
- **PWA Support** for offline play

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
